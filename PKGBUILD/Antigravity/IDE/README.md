# Antigravity IDE PKGBUILD

## Description
This PKGBUILD packages the Google Antigravity IDE, a standalone, AI-first integrated development environment built on top of VS Code.

## Installation Targets
* **Downloads**: Precompiled `Antigravity IDE.tar.gz` from Google Release and the application SVG icon.
* **Installs**: 
  * Full IDE application tree to `~/.local/share/Antigravity IDE/`
  * Launcher symlink to `~/.local/bin/antigravity-ide`
  * System icon to `~/.local/share/icons/`
  * Desktop menu entry to `~/.local/share/applications/`

## Testing
* Tested on CachyOS.
