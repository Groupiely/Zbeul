# Antigravity CLI PKGBUILD

## Description
This PKGBUILD packages the Google Antigravity CLI companion tool (`agy`), which is a lightweight, terminal-based interface for agent interaction.

## Installation Targets
* **Downloads**: Precompiled `cli_linux_x64.tar.gz` from Google Cloud Storage and SVG/PNG icon assets.
* **Installs**: 
  * CLI binary to `~/.local/bin/agy`
  * System icons to `~/.local/share/icons/`
  * Desktop menu entry to `~/.local/share/applications/`

## Testing
* Tested on CachyOS.
